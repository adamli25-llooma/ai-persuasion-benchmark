# ai-persuasion-benchmark
An empirical study examining how large language models (Claude, Gemini, GPT) deploy classical and modern persuasion techniques — including logos, pathos, ethos, scarcity, and authority — across varying prompt categories (silly, niche, ethically questionable, factually false). Completed for capstone project.
